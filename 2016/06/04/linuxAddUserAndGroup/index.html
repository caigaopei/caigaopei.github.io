linux添加组员相关命令

  • linux

/etc/group 存储当前系统中所有用户信息

Group: X : 123 : abc,def,xyz
组名 : 组密码占位符 : 组编号: 组中的用户类表
PS: 系统用户”root”组号一定是”0”,组号1~499为系统预留组编号.
用户手动创建的组是 >=500

/etc/gshadow 存储当前系统中用户组的密码信息

Group : * : : abc,def,xyz
组名称: 组密码 : 组管理者 : 组中用户名列表

/etc/passwd 存储当前系统中所有用户的信息

user : x : 123 : 456 : xxxxxxxxx :/home/user : /bin/bash
用户名 : 密码占位符: 用户编号: 用户组编号: 用户注释信息: 用户主目录 :shell类型

/etc/shadow 存储当前系统中所有用户的密码信息

bin:*:15980:0:99999:7:::
用户名: 密码 :::::

用户和用户组的基本命令

groupadd sexy –添加用户组 添加到/etc/group
groupmod -n(要修改的用户组名称) market(新组名称) sexy(源)
groupmod -g 668 market –更改组编号
groupadd -g 888 boss –直接创建用户组并且指定组编号
groupdel market –删除用户组
useradd -g sexy sdf(用户名:史蒂夫)
useradd -d /home/xxx cjk (创建用户苍井空,指定用户初始文件夹)
用户添加备注
usermod -c dgdzmx(新) sdf

修改用户名
usermod -l cls(新) sdf

修改用户个人文件目录
usermod -d /home/cls cls

切换用户组
usermod -g sexy(组) sdf(用户)

删除用户
userdel -r(循环) sdf

除了root以外,其他用户无法登陆系统
touch /etc/nologin (内容无关紧要)

用户和用户组的进阶命令

锁定用户账号
passwd -l cls(用户名)
解锁账户
passwd -u cls(同上)
清除账号密码
passwd　-d　cls(同上)
将用户添加到附属组
gpasswd -a cls(用户名) boss(组名)
临时切换用户组
newgrp boss
将用户踢出附属组
gpasswd -d cls boss
新建用户添加到主要组合附属组
useradd -g boss(主要组) -G group1,group2(附属组)....
添加组密码
gpasswd boss(组名)

其他命令

切换用户命令
su username
切换到root
su -
